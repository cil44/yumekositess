# Yumeko Bot Deployment Guide

Follow these steps to upload your code to GitHub and deploy it to Cloudflare Pages.

## 1. Upload to GitHub

Open your terminal in this project folder and run these commands one by one:

```bash
# Initialize git
git init

# Add all files
git add .

# Commit the files
git commit -m "Initial commit: Yumeko Bot Website"

# Set branch to main
git branch -M main

# Connect to your GitHub (REPLACE the URL below with your actual repo URL)
# Create a repo first at https://github.com/new
git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git

# Push to GitHub
git push -u origin main
```

## 2. Deploy to Cloudflare Pages

1. Go to [Cloudflare Dashboard](https://dash.cloudflare.com/).
2. Navigate to **Workers & Pages** > **Create application** > **Pages** > **Connect to Git**.
3. Select your GitHub repository.
4. Use these **Build Settings**:
   - **Framework preset**: `Vite`
   - **Build command**: `npm run build`
   - **Build output directory**: `dist`
5. Click **Save and Deploy**.

## 3. Environment Variables (If needed)

If you use the Gemini AI features, add this in Cloudflare Pages settings:
- Variable Name: `GEMINI_API_KEY`
- Value: `your_api_key_here`
